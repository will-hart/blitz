blitz.ui
========

Provides a PySide (Qt) backed UI for the Blitz Data Logger

 - :mod:`blitz.ui.dialogs` - provides dialogs for interacting with the logger
 - :mod:`blitz.ui.mixins` - provides code for updating the UI to match logger state
 - :mod:`blitz.ui.widgets` - provides widgets for displaying logger information
 - :mod:`blitz.ui.windows` - provides code for updating the UI to match logger state
